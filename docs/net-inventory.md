  `VLLM_NO_USAGE_STATS=1`, `DO_NOT_TRACK=1`.
- `podman/docker pull` (container image). Disposition: **lease**.
- `run-reranker.sh` model download. Disposition: **broker + lease**.

LAN peers. Disposition: **loopback or overlay** (amended B-3), routed through
the thin local-client helper so B-5 token auth lives in one place.
